## **Inflation & Purchasing Power: A Cross-Country Analysis**

An exploratory data analysis investigating the relationship between inflation and GDP per capita across five economies: India, United States, Germany, Netherlands, and Türkiye.

### Overview
This project examines whether higher-income countries experience lower inflation, and how economic capacity affects the real impact of inflation on purchasing power. A simple linear regression model is applied alongside time-series and cross-sectional analysis.

### Data Source
World Bank Open Data — Inflation (consumer prices, annual %) and GDP per capita (current US$)

### Tools Used
Python, pandas, matplotlib, Google Colab

### Key Findings
- Higher-income countries (Germany, Netherlands, US) show lower and more stable inflation
- GDP per capita alone is a weak predictor of inflation (R² ≈ 0.25)
- Türkiye is a significant outlier, demonstrating that inflation can escalate independently of income level
- Inflation should be interpreted as a relative economic pressure, not an absolute metric

### Limitations
Small sample size (5 countries) and the presence of an outlier limit generalisability. Further analysis with larger datasets and additional variables would yield more robust conclusions.
